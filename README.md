Le projet est un interpreteur Befunge depuis le navigateur il comprends donc un .css .js .html
voici un lien utile qui comprend entre autre les differente instruction possible afin de tester:
https://esolangs.org/wiki/Befunge 

Je conseil de charger le fichier text pour tester, puis de tester d'abord en mode debug, pour se faire :
load... puis debug puis presser entrer pour continuer (rester appuyé pour aller vite)

le programme va alors ecrire un "v" à une certaine case, puis print un hello world en utilisant la meme mécanique d'écrituredans une case, il va ensuite pusher des nombre en faisant certaine opérations dessus, on atteint ensuite un aiguilleure aléatoire,on push un 0 ou un 1 fonction du haut ou du bas pui aiguilera sur un if vertical, si on part à droite, on saute une ligne grace au code ascii de \n on demende ensuite à l'utilisateur de saisi 2  nombre qu'on additione, on les print, et on print un "thanks for testing !"
